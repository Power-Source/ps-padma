# Willkommen bei PS Padma

Schön, dass du da bist.  
PS Padma ist dein visueller Theme-Builder für WordPress und ClassicPress — schnell, flexibel und ohne unnötigen Ballast.

## Warum PS Padma?

- Visueller Editor mit Drag-and-Drop-Grid
- Design-Anpassung per Point-and-Click
- Viele integrierte Blöcke und Layout-Möglichkeiten
- Keine Pflicht zu externen Builder-Plugins
- Fokus auf Performance, Workflow und Kontrolle

## Quick Start

1. Theme installieren und aktivieren.
2. Im Admin-Bereich die PS Padma Seiten öffnen.
3. Visual Editor starten.
4. Layout bauen, Blöcke setzen, Styles anpassen.
5. Fertig — veröffentlichen und weiter optimieren.

## Empfohlene Server-Einstellungen

Für eine entspannte Arbeit im Editor empfehlen wir:

- PHP 8.0 oder höher
- `post_max_size` mindestens `32M`
- `memory_limit` mindestens `256M`

## Sicherheit

PS Padma kann in bestimmten Blöcken PHP verarbeiten. Wenn du das deaktivieren möchtest, setze in deiner `wp-config.php`:

    define('PADMA_DISABLE_PHP_PARSING', false);
